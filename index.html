Toit is a free and open source object-oriented language designed specifically for IoT

Getters and Setters

In Toit, getters and setters have the same syntax (for users of a class) as public member variables. Therefore there is no need to write trivial setters and getters.

Learn more...
class UnneededPrivacy:
  // No point in making this private since
  // it has a trivial getter and setter.
  x_ /int  // Private member variable.

  constructor x/int:
    x_ = x

  // Getter.
  x -> int:
    return x_
